# نظام بحث عن شهادات الطلاب

##  مدرسة أسامة بن زيد الاعدادية بنين

---

###  وصف التطبيق
تطبيق ويب للبحث عن شهادات الطلاب باستخدام الرقم الشخصي.
- **صفحة المستخدم**: إدخال الرقم الشخصي وعرض الشهادة (PDF أو صورة)
- **صفحة الأدمن**: رفع ملف PDF واحد يحتوي جميع الشهادات وفهرستها تلقائياً

---

###  التقنيات المستخدمة
| التقنية | الغرض |
|---------|-------|
| Node.js | بيئة التشغيل |
| Express | خادم الويب |
| pdfjs-dist | قراءة PDF وفهرسته |
| pdf-lib | استخراج صفحات من PDF |
| multer | رفع الملفات |
| express-session | جلسات تسجيل الدخول |

---

###  هيكل المشروع
```
server/
├── index.js           # ملف السيرفر الرئيسي
├── package.json       # إعدادات المشروع
├── public/
│   ├── index.html     # صفحة المستخدم
│   ├── admin.html     # صفحة الأدمن
│   └── style.css      # التنسيقات
├── uploads/           # مجلد رفع PDF والفهرس
│   ├── certificates.pdf
│   └── index.json
└── node_modules/      # الحزم (لا ترفع إلى GitHub)
```

---

###  APIs

| المسار | الطريقة | الوصف |
|--------|---------|-------|
| `/api/login` | POST | تسجيل دخول الأدمن |
| `/api/logout` | POST | تسجيل خروج |
| `/api/check-auth` | GET | التحقق من حالة الدخول |
| `/api/upload` | POST | رفع ملف PDF (أدمن فقط) |
| `/api/search` | POST | بحث برقم شخصي (يعيد PDF) |
| `/api/status` | GET | حالة المرفوعات |

---

###  تشغيل التطبيق محلياً
```bash
cd server
npm start
```
ثم افتح `http://localhost:3000`

###  بيانات الدخول للأدمن
- **اسم المستخدم**: `admin`
- **كلمة المرور**: `admin123`
> (يمكن تغييرها في ملف `index.js` السطر 12-13)

---

###  النشر على Render
1. ارفع المشروع إلى GitHub (ملاحظة: ملف `node_modules` لا يرفع)
2. في Render.com:
   - New Web Service → Connect GitHub
   - Build Command: `npm install`
   - Start Command: `node index.js`
   - Environment Variable: `NODE_VERSION = 24.16.0`

---

###  كيفية الاستخدام
1. **الأدمن**: ادخل على `/admin`، سجل دخول، ارفع ملف PDF للشهادات
2. **المستخدم**: اكتب الرقم الشخصي ← اضغط "عرض الشهادة"
3. **تحميل الشهادة**: بعد ظهورها، استخدم "تحميل PDF"
4. **حفظ كصورة**: استخدم "حفظ كصورة" ← اضغط مع الاستمرار للحفظ
